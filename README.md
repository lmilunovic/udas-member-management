# UDAS Member Management

Organization member management system with a Spring Boot backend and frontend application.

## Project Structure

```
├── backend/          # Spring Boot REST API (Java 17+, PostgreSQL)
├── frontend/         # Frontend application (coming soon)
└── opencode.json     # OpenCode configuration with superpowers
```

## Prerequisites

- **Backend**: Java 17+, PostgreSQL 14+, Maven
- **Frontend**: Node.js 18+ (when implemented)

## Getting Started

### Backend

See [backend/README.md](backend/README.md) for detailed setup instructions.

```bash
cd backend
./mvnw spring-boot:run
```

API available at `http://localhost:8080`

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Spring Boot 3.x, Java 17 |
| Database | PostgreSQL 14+ |
| Migrations | Flyway |
| API | REST with OpenAPI |
| Frontend | TBD |
